工具确认协议测试

测试说明

此页面用于验证工具确认协议是否正常工作。请按照以下步骤进行测试：

  1. 在浏览器中打开 http://localhost:5174/
  2. 点击右上角的"连接"按钮
  3. 在配置中设置服务器地址为: ws://localhost:8081
  4. 点击"连接"按钮连接到测试服务器
  5. 等待5秒，测试服务器将自动发送工具确认请求
  6. 在消息列表中查看工具确认消息
  7. 测试批准和拒绝按钮

预期结果

工具确认消息应显示：

工具名称：execute_command

描述：构建项目

参数：
command: "npm run build"
requires_approval: true

按钮：应显示"批准"和"拒绝"按钮

第二个工具确认消息（10秒后）：

工具名称：read_file

描述：读取主文件内容

参数：
path: "src/main.ts"

测试验证

请检查以下功能是否正常工作：

工具确认消息正确显示在消息列表中

工具名称、描述和参数正确显示

批准按钮正常工作（点击后消息状态更新）

拒绝按钮正常工作（点击后消息状态更新）

第二个工具确认消息在10秒后正确显示

服务器正确接收并响应工具确认响应

测试服务器状态

测试服务器运行在：ws://localhost:8081

开发服务器运行在：http://localhost:5174/

如果测试服务器没有响应，请检查终端输出。
